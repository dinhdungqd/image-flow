2 Data

Data We Access & Why

The extension only accesses information that is directly required by a feature you explicitly activate. Nothing is collected passively in the background.

  • 🌐 Active tab URL — detected to identify when you open a Google Labs project page, so the extension can sync the correct project ID. The URL is never stored on remote servers.
  • 📸 Tab screenshots (crop tool) — a screenshot of the visible tab is captured only when you explicitly trigger the crop tool (Alt+C or the in-panel button). The cropped image is saved locally as a reference image and is never transmitted to our servers.
  • 🖼️ Images via right-click menu — when you choose "Save to Image Flow" from the context menu, the image URL or base64 data is saved locally to your gallery. This happens only on your explicit action.
  • 📝 Prompt library content — prompts you create and save are stored entirely in chrome.storage.local on your device. They are never uploaded without your action.
  • ⚙️ Extension settings — preferences (widget toggle, slash command, Telegram credentials you enter, etc.) are stored locally and used only to configure extension behavior.
⚠️

We do not collect: names, email addresses, IP addresses, browsing history, keystrokes, form inputs, passwords, or any personally identifiable information beyond what you explicitly type into the extension's own settings fields.

3 Permissions

Chrome Permissions Explained

Every permission requested by Image Flow has a specific, limited purpose. Here is a plain- English explanation of each one:

Permission Purpose
activeTab Read the URL of the currently active tab to detect Google Labs projects.
tabs Monitor tab navigation to track the active project and focus the correct tab when needed.
scripting Inject the crop-selection overlay and automation scripts into Google Labs pages.
storage Save prompts, reference images, project data, and settings locally in your browser.
downloads Save generated images to your Downloads folder with custom filenames you define.
notifications Show desktop notifications for task completions and available extension updates.
sidePanel Render the extension UI in Chrome's native side panel.
identity Handle OAuth2 flow for optional Google Drive backup — only when you choose to enable it.
contextMenus Add "Save to Image Flow" options to the browser right-click menu on images.
alarms Schedule a version-check request every 6 hours to notify you of updates.
4 Storage

Local Data Storage

All user-generated data — including your prompt library, reference images (stored as base64), project configurations, and usage counters — is saved exclusively in chrome.storage.local on your own device.

  • 🔒 Never leaves your device unless you explicitly activate the Google Drive backup feature.
  • 🗑️ Automatically cleared when you uninstall the extension from Chrome.
  • 🛠️ Manually deletable at any time via Chrome DevTools → Application → Storage → Extension storage.
5 Third-Party

Third-Party Services

The extension may communicate with the following external endpoints. All connections are triggered by your direct actions — there are no background analytics calls.

  • 🔔 nguyenanhdung.io.vn/api/version — a lightweight GET request (no personal data) is sent every 6 hours to check whether a newer version of the extension is available.
  • ☁️ Google OAuth2 & Google Drive — used only if you enable the Drive backup feature. Authentication is handled entirely by Google's own OAuth2 flow. The extension requests only the drive.file scope (files it creates), not full Drive access.
  • ✈️ Telegram Bot API — used only if you enter your personal bot token and chat ID in settings. Messages are sent to your own bot. Credentials are stored locally on your device and never sent to our servers.
  • 🎨 CivitAI API (api.civitai.com) — accessed only when you explicitly use the model-search feature inside the extension panel.
  • 🌐 Google Translate API — used only when you trigger the optional prompt-translation feature. No data is stored by us.
✅

Image Flow does not use any advertising networks, third-party analytics platforms (Google Analytics, Mixpanel, etc.), or crash-reporting services. Zero telemetry.

6 Your Rights

Your Controls & Rights

You are in full control of your data at all times.

  • 🎛️ Disable individual features — the floating widget, slash-command overlay, and Telegram notifications can each be toggled off independently in the Settings tab.
  • 🗑️ Delete all local data — uninstall the extension, or clear its storage manually via Chrome DevTools → Application → Extension Storage.
  • 🔗 Revoke Google Drive access — visit myaccount.google.com/permissions at any time to revoke the extension's Drive authorization.
  • 👤 No account required — Image Flow works fully offline and without any sign-in, except for the optional Google Drive feature.
7 Children

Children's Privacy

Image Flow is not directed at, and does not knowingly collect information from, children under the age of 13. If you believe a child has used this extension in a way that may have collected personal information, please contact us so we can take appropriate action.
